# App Icon Assets

- Primary source asset: `app-icon-mymind-ios-2024.png`
- Visual source: the official `mymind` iOS App Store artwork for `mymind – Extend your mind`
- Source URL: `https://is1-ssl.mzstatic.com/image/thumb/Purple221/v4/2e/57/10/2e571035-f756-32fb-d29d-52f9f40f84be/AppIcon-Release-1x_U007epad-0-1-sRGB-85-220-0.png/1024x1024bb.jpg`
- Selection rationale: the App Store release notes for version `3.2` on `2024-04-03` say "our little thinking guy is now our primary app icon"
- Reason for keeping a local master: generate the Chrome extension icons and macOS app icon set from one 1024x1024 source instead of upscaling `icon-128.png`
- Intended use: personal local use for this repository only

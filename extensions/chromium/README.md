# Cosmogony Chromium Companion

This extension is intentionally thin:

- capture current tab context
- capture selected text from the context menu
- handshake with the macOS app over `127.0.0.1`
- forward page or clipboard payloads

It does not include:

- QuickDock
- sidebar UI
- browser manager UI
- cloud sync

Load it unpacked from the `extensions/chromium` folder.
